# Report Figures

This folder contains the final, reviewer-facing figures for the ICU length-of-stay project.

The plotting code that produced these figures is kept separately in `../visualization_scripts/` so this folder can stay focused on the visual outputs.

## Included Figures

- `binary_model_comparison.png` - comparison of binary classification model metrics
- `binary_roc_curve.png` - ROC curve for the binary short-stay task
- `top_feature_importances.png` - top Random Forest feature importances
- `confusion_matrix_comparison.png` - confusion matrix comparison across models
- `binary_vs_three_way_target_setup.png` - target setup comparison for binary and three-way LOS classification
- `los_class_mix_by_icu_unit.png` - LOS class mix across ICU units
